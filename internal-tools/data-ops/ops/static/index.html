Trust Halal · data-ops

desktop only · direct to prod

Backfill field from Google

Dry run resolves candidates and logs what it would do without calling Google or writing anything. Uncheck to actually resync + backfill. Only Google-linked places whose field is NULL are touched (never clobbers existing values).

Import hero image from Google

Only touches Google-linked places with zero photos — pure gap-fill, never clobbers an owner's curated photo. Pulls the listing's headline photo via the official Google Place Photos API, stores it as the hero (source GOOGLE), and records Google's required attribution in the caption.

Sync Google data (rating · hours · website)

Refreshes volatile Google fields (rating, review count, opening hours, and google_synced_at) for every Google-linked place; website fills only if empty. Meant to run weekly/biweekly. Note: these fields are Google's pricier SKU tiers, so a full run has real API cost — throttle and mind the cadence.
